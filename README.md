# Dead-Mans

Интерактивная панель управления для стрима и зрительских ивентов.  
Проект позволяет вести игру с командами, считать очки, применять модификаторы от зрителей и управлять ходом раунда в реальном времени.

---

## Основные возможности (задумка проекта)

- **Управление игрой**: лоадауты, раунды, состояния ячеек.
- **Таблица лидеров**: команды, очки, штрафы, история изменений.
- **Модификаторы от зрителей**: бонусы/штрафы, влияющие на баланс игры.
- **Панель управления стримера**: быстрые действия (сброс, пауза, старт/стоп и т.д.).
- **Будущее развитие**:
  - авторизация через Twitch и роли (стример, модераторы и др.);
  - поддержка нескольких игр в одном проекте;
  - генерация и хранение составных изображений (например, итоговый экран раунда);
  - удобная интеграция со стример‑ботом.

---

## Структура репозитория

- `legacy-v1` — старый прототип (vanilla JS), используется как источник идей и логики.
- `frontend` — SPA на React + TypeScript (Vite, React Router, React Query, MUI, i18n) с feature-oriented структурой и слоистым app shell.
- `backend` — ASP.NET Core Web API с базовой layered architecture: `Application`, `Domain`, `Infrastructure`, `Controllers`, `Data`.
- `STACK.md` — краткое описание выбранного стека и используемых технологий.
- `CONTEXT.md` — живой контекст проекта и история решений.

## Быстрый старт (frontend)

```bash
cd frontend
npm install    # один раз, для установки зависимостей
npm run dev    # запуск dev-сервера
```

После запуска открой URL, который покажет Vite (обычно `http://localhost:5173` или `5174`).
